## ファイル構成

### run_once_


- 0x
  - 他のソフトをインストールするのに必要なものが入っている
  - git, cmake, curlなど

- 1x
  - プログラミング言語をインストールする
  - python, go, c++など
- 特定のツールにのみ依存している言語は、そのツールの依存関係としてインストールする(nodejsなど)

- 2x
  - シェル(zsh)やそのプラグインなど

- 3x
  - エディタ(neovim)やそのプラグインなど

- 4x
  - 便利なツールが入っている
  - direnv, exa, bat, chezmoiなど
